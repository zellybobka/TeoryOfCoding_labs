# TheoryOfCoding_labs

Небольшой учебный репозиторий с двумя лабораторными работами по теории кодирования на Python.

## О проекте
В этих работах я реализовал базовые алгоритмы кодирования и декодирования данных, включая обработку ошибок при передаче.  
Проекты включают:
- построение и проверку параметров линейных кодов;
- генерацию рабочих матриц и кодирование сообщений;
- поиск и исправление ошибок при декодировании;
- реализацию BCH-кода и работу с конечными полями.

## Что это показывает
Этот репозиторий демонстрирует:
- умение переводить математическую задачу в код;
- работу с алгоритмами и структурами данных;
- аккуратную реализацию вычислительной логики в Python;
- понимание принципов надёжной передачи данных.
